TOKEN RULES (CRITICAL):
1. For recipient name, ALWAYS use {{nombre}}.
2. STATIC URLs: If user provides a specific URL, use it VERBATIM. Never replace with a variable.

4. Do not leave generic placeholders like [Link] or [Date] — invent logical data or use allowed tokens.
